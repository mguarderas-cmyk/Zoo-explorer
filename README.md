# Zoo Explorer

Guardianes de la biodiversidad: conectando naturaleza y tecnologia para un futuro sostenible.

Zoo Explorer es una plataforma educativa para convertir las visitas a la Zoobotanica de Eight Academy en experiencias de observacion, descubrimiento y accion. La primera version del repositorio incluye una pagina web funcional, una bitacora local de observaciones y una estructura lista para crecer hacia una aplicacion completa.

## Estado actual

- Pagina web estatica funcional en `index.html`.
- Estilos principales en `styles.css`.
- Interactividad y bitacora local en `app.js`.
- Documentacion del proyecto en `docs/`.
- Recursos visuales y multimedia en `assets/`.

## Como abrir la pagina

Abre `index.html` en un navegador. La pagina funciona sin instalar programas adicionales.

## Problema

La Zoobotanica de Eight Academy alberga mas de 38 especies documentadas de animales y plantas, pero muchos estudiantes la visitan sin conocer realmente las especies que tienen frente a si, sus roles ecologicos o sus cuidados.

Esta brecha genera riesgos de descuido involuntario, perdida de oportunidades de aprendizaje experiencial y menor conciencia ecologica dentro de la comunidad educativa.

## Propuesta

Zoo Explorer organiza la experiencia en tres momentos:

1. Observar: reconocer especies y registrar evidencias.
2. Descubrir: aprender datos relevantes sobre plantas, animales y ecosistemas.
3. Actuar: completar misiones de cuidado, monitoreo y conservacion.

## Ruta sugerida

1. Publicar esta primera pagina con GitHub Pages.
2. Agregar imagenes reales de las especies del campus en `assets/`.
3. Completar una base de datos inicial de especies.
4. Convertir la pagina estatica en una PWA.
5. Integrar reconocimiento de imagenes y usuarios cuando el prototipo este validado.

## Tecnologia planificada

La vision del proyecto contempla React, Tailwind CSS, Node.js, Express, MongoDB, servicios de IA para reconocimiento y contenido educativo, y una futura capa de insignias digitales. Esos componentes se iran incorporando por etapas.
