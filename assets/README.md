# Recursos visuales

Esta carpeta debe guardar imagenes, iconos, codigos QR y fotografias de especies del campus.

## Estructura recomendada

- `qr-zoo-explorer.png`
- `species/`
- `logos/`
- `screenshots/`

Para la pagina web, las imagenes de especies pueden agregarse luego en `assets/species/` y enlazarse desde `app.js`.
